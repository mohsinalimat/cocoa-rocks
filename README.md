## Build

``` bash

# install deps
npm install

# create dist build
npm run dist

# run in watch mode with broweser reload (on http://localhost:3333)
npm run serve

# enrich the data file with information from GitHub
export COCOA_ROCKS_GITHUB_TKN=<<github-tkn>> || set COCOA_ROCKS_GITHUB_TKN=<<github-tkn>>
npm run updatedata

```
